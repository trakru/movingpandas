Welcome to MovingPandas's documentation!
========================================

MovingPandas implements trajectory classes and functions built on top of GeoPandas.

The project home is: http://movingpandas.org

You can find extensive usage examples in the tutorial notebooks included in the repository and on MyBinder:

* Tutorial_0_: Getting started
* Tutorial_1_: Ship data analysis
* Tutorial_2_: Bird migration analysis
* Tutorial_3_: Horse collar data exploration
* Tutorial_4_: Trajectory aggregation (flow maps)
